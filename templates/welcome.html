How to Use This Site

  1. Click the notebook name to open it in JupyterLab. The notebook will run entirely in your browser (no server required).
  2. Read the markdown cells for theory and explanations, then run the code cells by selecting them and pressing Shift+Enter.
  3. Experiment! Modify parameters, change initial conditions, and re-run cells to see how the system responds.

JupyterLab Tips

  1. Run a cell: Shift+Enter runs the current cell and moves to the next. Ctrl+Enter runs in place.
  2. New cell: Click the + button in the toolbar, or press B (below) / A (above) in command mode.
  3. Cell type: Change between Code and Markdown from the toolbar dropdown.
  4. Kernel: Each notebook runs a Python kernel in your browser via Pyodide. If a notebook seems stuck, use Kernel → Restart Kernel from the menu.
  5. Reset: To reset all outputs, use Run → Restart Kernel and Run All Cells.
